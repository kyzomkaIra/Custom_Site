Про мене

Я Кузьомка Ірина мені 17років. Я навчаюся у Вищому професійному училищі №1, яке знаходиться в м.Рівне.

Монітор
Монітор — це пристрій для візуального відображення інформації.
Клавіатура
Клавіатура — це пристрій введення текстової інформації.
Мишка
Мишка — це пристрій для керування курсором на екрані.
Принтер
Принтер — це пристрій для друку тексту та зображень на папері.
Сканер
Сканер — це пристрій для оцифровування зображень та тексту.
Веб-камера
Веб-камера — це пристрій для передачі відео через інтернет.
Графічний планшет
Графічний планшет — це пристрій для цифрового малювання.
Колонки
Колонки — це пристрої для відтворення звуку.
Мікрофон
Мікрофон — це пристрій для запису звуку.
